# Quiz App

- Desenvolvido com ReactJs e Next.js com TypeScript consumindo a Api disponível em:
  https://test-quiz-app-backend.herokuapp.com

## Instalação

```sh
# Instalando as dependências
yarn install

# Habilita o husky para validação ao commitar
yarn husky install

# Starta o projeto
yarn dev
```
